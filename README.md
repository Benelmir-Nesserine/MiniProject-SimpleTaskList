# Simple Task Manager

This project is a command-line task manager built in Python.
(A simple command-line task manager built with Python and JSON persistence)

## Features

-   Add new task
-   View all tasks
-   Mark task as done
-   Save tasks to tasks.json
-   Reload tasks from tasks.json

## Data Storage

Tasks are stored in a JSON file called `tasks.json`. Each task
contains: - title (string) - done (true/false)

## How to Run

Make sure Python 3 is installed, then run:

python tasks.py
